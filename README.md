# Scala Vault

A secure and light weight wallet to store your Scala coins on any Android device.

## Features		
* Multiple Wallets
* Subaddress Support
* Stealth Mode
* QR Code Scanning
* Automatic Price Conversion to other Currencies
* Easy to use UI
